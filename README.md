# Number Analysis Program (C)

## Description
A C program that performs statistical analysis on a set of user-entered integers. It calculates key metrics such as sum, average, maximum, and minimum values.

## Features
- Accepts multiple integer inputs
- Calculates sum and count
- Finds maximum and minimum values
- Computes average of entered numbers

## Technologies Used
- C Programming Language
- Arrays
- Loops

## How to Run
1. Compile the program using a C compiler.
2. Execute the program.
3. Enter the number of integers and their values as prompted.

## Learning Outcome
- Gained experience with array manipulation
- Learned basic statistical computation using C
